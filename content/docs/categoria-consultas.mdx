## Criando e Gerenciando Categorias  

Existem duas maneiras de gerenciar e criar categorias no Suryamed: **dentro de uma consulta** ou **na listagem de consultas**.  

### Criando Categorias Dentro de uma Consulta  

1. Durante a consulta, clique no botão **"Categorias de Consulta"** para acessar as opções de categorias.  
   
<Image  
  src="/images/adicionar-categoria-consulta.jpg"

  width={1000}

/>  

2. Para adicionar uma categoria à consulta, clique na categoria desejada. Caso precise criar uma nova categoria, clique no botão **"+"** para adicionar uma categoria personalizada.  

<Image  
  src="/images/categorias-de-consulta.png"

/>  

> **Nota**: Categorizar suas consultas ajuda a manter tudo organizado e facilita o uso de filtros na listagem de consultas.  

### Gerenciando Categorias na Listagem de Consultas  

Você também pode criar, editar ou gerenciar categorias diretamente na página de **listagem de consultas**.  

1. Acesse a página de listagem de consultas.  
2. Clique no botão **Categorias**.
3. Clique no ícone de **lápis** para editar uma categoria existente.  
4. Para criar uma nova categoria, clique no botão **"Nova Categoria"**.

Categorizando suas consultas, você melhora a organização e pode usar filtros para buscar consultas específicas de forma rápida e eficiente.
